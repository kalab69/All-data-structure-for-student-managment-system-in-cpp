# All-data-structures-for-student-managment-system-in-cpp
Student Information Management System
Overview
This project is a Student Information Management System implemented in C++ using different data structures.
It demonstrates how student records can be managed using:


Singly Linked List


Doubly Linked List


Stack


Queue


Binary Search Tree (BST)


The program is completely menu-driven and runs in the terminal.

Features
1. Singly Linked List


Insert student data at the beginning


Delete student data from the beginning


Display all student records


Search student by ID



2. Doubly Linked List


Insert student data at the beginning


Delete student data from the beginning


Display records backwards


Search student by ID



3. Stack


Push student data


Pop top student data


Peek top student


Search student by ID



4. Queue


Enqueue student data


Dequeue student data


Peek front student


Display all students


Search student by ID



5. Binary Search Tree (BST)


Insert student data


Delete student data


Search student by ID


Tree traversals:


Inorder


Preorder


Postorder
